Some AWS cli commands for IAM
*********************************
aws iam list-users

aws iam list-groups

aws iam create-group --group-name Admins

aws iam create-user --user-name Bob

aws iam create-login-profile --user-name Bob --password redhat@123

aws iam update-login-profile --user-name Bob --password newpwd@123

aws iam update-user --user-name Bob --new-user-name Robert

aws iam add-user-to-group --user-name Bob --group-name Admins

aws iam delete-group --group-name MyTestGroup

aws iam delete-user --user-name Bob

aws iam attach-user-policy --policy-arn arn:aws:iam::aws:policy/AdministratorAccess --user-name Bob

aws iam list-attached-user-policies --user-name Bob

aws iam help ........to get any help
